**Objective**: To analyzes and visualizes the performance of Nifty 50 stocks using historical stock market data. The dashboard provides insights into stock price movements, trading volume trends, and stock correlations through interactive visualizations built with Streamlit and Power BI.
**Tools used:** Python|Pandas|Power BI
**Features:** Stock Analysis Sector, Price, Stock Correlation Matrix Analysis |Interactive filters and Visualizations |Power BI Dashboards.
**Conclusion**: The project helps investors and analysts understand stock performance through interactive dashboards, enabling data-driven investment decisions.
